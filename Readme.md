# ProductListScreen App

This is a React Native application that displays a list of product cards fetched from the [Fake Store API](https://fakestoreapi.com/). Each card includes a product image, title, price, a buy button, and action icons (like, refresh).

---

## Features

- Fetches products from `https://fakestoreapi.com/products`
- Displays products using a `FlatList` component
- Each product card includes:
  - Image
  - Title
  - Price
  - Buy button
  - Action icons (like, refresh)
- Responsive UI for mobile devices
